# AWS CloudFormation Infrastructure as Code

## Project Overview

This project demonstrates Infrastructure as Code (IaC) using AWS CloudFormation.

The template automates the deployment of:

- VPC
- Subnet
- Security Group
- EC2 Instance

---

## Deliverables

### Working CloudFormation Template

- YAML-based Infrastructure as Code
- Reusable deployment template
- Parameterized configuration

### Infrastructure Deployment

- VPC Creation
- EC2 Deployment
- Security Configuration
