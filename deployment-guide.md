# CloudFormation Deployment Guide

## Objective

Deploy AWS infrastructure using Infrastructure as Code (IaC).

## Resources Created

- VPC
- Subnet
- Security Group
- EC2 Instance

## Deployment Steps

1. Open AWS CloudFormation Console.
2. Create Stack.
3. Upload cloudformation-template.yaml.
4. Specify stack name.
5. Review resources.
6. Deploy stack.

## Outputs

- VPC ID
- EC2 Instance ID

## Benefits

- Repeatable deployments
- Automated infrastructure creation
- Reduced manual configuration
